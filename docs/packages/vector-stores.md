# Vector Stores 设计

## 文档定位

本文档用于说明 `@rag-sdk/vector-stores` 在目标架构中的边界。

## 目标架构边界

目标架构中，RAG SDK 不负责具体向量存储能力适配。

向量数据库、搜索引擎或其他索引存储属于外部系统能力，应由业务侧或外部实现自行接入。

## 不继续扩展的职责

`vector-stores` 不作为目标架构中的核心对外能力展开。

SDK 不在该子包中沉淀具体向量数据库适配。

SDK 不把某个向量数据库作为默认存储实现或架构前提。

如果仓库中继续保留该子包，应只用于历史兼容、实验验证或边界迁移说明，不应作为新增能力继续扩展。
